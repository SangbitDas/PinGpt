# ChatGPT Chat Pinner

A Chrome extension to save and organize your important ChatGPT conversations.

## Why We Built This

ChatGPT doesn't provide a way to bookmark or save specific conversations for quick access. As your chat history grows, finding important conversations becomes tedious. This extension lets you **pin chats directly from ChatGPT** and access them anytime through the extension.

## Impact

- **Never Lose Important Chats**: Save conversations you want to revisit without scrolling through history.
- **Organized Access**: Group pinned chats into custom categories for better management.
- **Cross-Device Sync**: Pins sync across all your Chrome browsers via Chrome sync storage.
- **One-Click Access**: Open any saved chat instantly from the extension popup or full page view.

## How to Use

### Installation

1. Download or clone this repository.
2. Open `chrome://extensions/` in Chrome.
3. Enable **Developer mode** (top right).
4. Click **Load unpacked** and select the extension folder.

### Pinning a Chat

1. Open [ChatGPT](https://chatgpt.com).
2. Hover over any chat in the sidebar â†’ click the **three-dot menu** â†’ select **Pin**.

### Viewing Pinned Chats

- Click the **extension icon** to see your pinned chats in a popup.
- Click **Full Page View** for a larger interface with search and category management.

### Managing Categories

- In Full Page View, click **Manage Categories** to create/delete categories.
- Use **+ Add** on any chat to assign it to a category.
